  </p>
  <div class="content">
    <p><b>Langkah-langkah:</b></p>
    <ul className="list-decimal space-y-3 pb-2 pl-10">
      <li>Buka XAMPP dan aktifkan MySQL dari control panel-nya.</li>
      <div class="space-y-3">
        <p class="rounded-xl w-full border border-zinc-200 dark:border-zinc-800">
